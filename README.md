# bestBuy.com-Clone

This project is the clone of bestBuy.com .Here you can Shop Best Buy for electronics, computers, appliances, cell phones, video games & more new tech.

## Features:
1. shop and buy
2. filter Product .
3. Discount Counter.
4. Order Summary.

## Tech stack used:
1. Reactjs
2. HTML
3. CSS
4. bootstrap
5. Git
6. JavaScript
7.
